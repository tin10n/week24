## Overview

This is a React Native Expense Tracker application that allows users to log their daily expenses.
Users can enter an expense description, amount, and category (such as Food or Transport).
All expenses are stored locally on the device using AsyncStorage, so data persists between app sessions.


### Features

- Add daily expenses
- Select expense categories (Food, Transport, Entertainment, Other)
- View a list of all logged expenses
- Edit an existing expense
- Data saved locally using AsyncStorage
